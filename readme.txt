Install & running server

1- run "npm install" to install dependecies
2- run "npm run start" to build and run the server at port 3000
3- url server will be: http://localhost:3000

-----------------------

An api to resize images endpoint is :

http://localhost:3000/api/images?filename=santamonica&height=500&width=600

query parameters are:

- filename (required): image file name and there is 5 images to test with names:
    encenadaport
    fjord
    icelandwaterfall
    palmtunnel
    santamonica

- height (required): should be a number 
- width (required): should be a number
